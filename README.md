# Prompt Redactor

Веб-приложение на React + Vite для удаления нужного фрагмента из текста.
Пользователь вставляет исходный текст, задаёт фрагмент для удаления и сразу получает результат.

## Возможности

- Ввод исходного текста
- Ввод текста, который нужно удалить
- Автоматический вывод результата
- Удаление первого вхождения или всех вхождений
- Опция чувствительности к регистру
- Копирование результата
- Светлая и тёмная тема
- Современный адаптивный UI (high-tech стиль)

## Технологии

- React 19
- Vite 7
- Tailwind CSS 4
- React Icons
- ESLint

## Требования

- Node.js `20+`
- npm `10+`

## Установка и запуск

```bash
npm install
npm run dev
```

После запуска откройте адрес из терминала (обычно `http://localhost:5173`).

## Скрипты

```bash
npm run dev      # запуск в режиме разработки
npm run build    # production-сборка
npm run preview  # локальный просмотр production-сборки
npm run lint     # проверка ESLint
```

## Как использовать

1. Вставьте текст в блок **Основной текст**.
2. Укажите фрагмент в блоке **Фрагмент для удаления**.
3. Настройте параметры удаления (все вхождения / регистр).
4. Скопируйте результат иконкой в блоке **Результат**.
